# Games-cart
An Ecom site for buying premium games.

## Features(Screens)

* Homepage
* Product Listing Page
* Wishlist
* Cart 
* Sign Up 
* Login

## Tech Stack

* HTML
* CSS
* Javascript

## To Run Locally

1. Clone the Repository
```
https://github.com/Rohitprasad83/games-cart.git
```
2. Go to Project Directory
```
cd games-cart
```
```
open the index.html
```
